# playwright-visual-testing

Project scaffold for visual testing using [Playwright](https://playwright.dev/) and [playwright-bdd](https://github.com/playwright-community/playwright-bdd).

## Setup

1. Install dependencies:
   ```bash
   npm install
   npx playwright install
   ```
2. Run tests (none included yet):
   ```bash
   npm test
   ```

This repository only contains initial configuration files and directory structure. Add feature files under `features/` and step definitions under `steps/` when you are ready to write tests.
